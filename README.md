# sales-data-analysis-python
Week-2 Internship Project
1. Project Overview
This project focuses on analyzing sales data using Python to extract meaningful business insights. The dataset contains sales transactions including order details, product information, customer data, pricing, and discounts. The goal is to calculate revenue, analyze product and category performance, identify monthly trends, and generate a summarized business report.

2. Objectives
* Create a structured sales dataset with at least 50 records.
* Calculate total sales amount and final payable amount after discount.
* Analyze product-wise and category-wise performance.
* Perform monthly sales analysis using order dates.
* Generate business insights for decision-making.
* Export the processed dataset as a CSV file.

3. Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- GitHub

4. Workflow
+ Data Creation
Generated random sales records using Python.
Ensured readable date format.
+ Data Processing
Calculated total amount.
Applied discount calculation.
Derived final amount.
+ Analysis
Overall revenue calculation.
Product-wise and category-wise performance.
Monthly sales trends.
Regional and payment mode analysis.
+ Output
Exported final dataset to CSV file.
